### Static

Allows you to specify the only direction the Prop will be placed in.

#### Parameters:

<ParameterList
  parameters={[
    {
      name: "Rotation",
      description:
        "integer, optional with default of 0. Can only have on of these four values: 0, 90, 180 or 270.",
    },
    {
      name: "Pattern",
      description:
        "Pattern slot. Used to locate a suitable position for the Prop and does not affect the Prefab’s direction.",
    },
  ]}
/>

### Pattern _(note: name of the Directionality Type)_

Allows the Prop to have the correct direction based on it environment. Allows you to link Prefab directions to Pattern assets for each direction.

#### Parameters:

<ParameterList
  parameters={[
    {
      name: "InitialDirection",
      description:
        "string with possible values: “N”, “S”, “E”, “W”. The direction the Props are facing in their original state. A balcony facing North in this context would mean that the balcony is looking North.",
    },
    {
      name: "NorthPattern",
      description:
        "Pattern slot. Defines environments where this Prop should place facing North.",
    },
    {
      name: "SouthPattern",
      description:
        "Pattern slot. Defines environments where this Prop should place facing South.",
    },
    {
      name: "EastPattern",
      description:
        "Pattern  slot. Defines environments where this Prop should place facing East.",
    },
    {
      name: "WestPattern",
      description:
        "Pattern slot. Defines environments where this Prop should place facing West.",
    },
    {
      name: "Seed",
      description:
        "string. Affects direction picking where more than one direction is possible.",
    },
  ]}
/>
